# Salary Prediction using Linear Regression

This project predicts employee salary based on years of experience using a Linear Regression model.

## Dataset
Salary dataset from Kaggle.

## Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn

## Workflow
1. Import libraries
2. Load dataset
3. Data visualization
4. Train-test split
5. Train Linear Regression model
6. Predict salary
7. Evaluate model using R² score

## Example Prediction
Predicted salary for 13 years experience ≈ 146,889
